# TP - C++ - Neural Network


Implementation d'un réseau de neurone simple. Création d'une bibliothèque élémentaire de manipulation de matrices, un csv reader et une bibliothèque simple de réseau de neurones simple pour résoudre le problème de reconaissance de chiffres écrit à la main.

## Pour compiler

```bash 
make 
./Neural_network
```
